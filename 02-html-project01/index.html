HTML & CSS Bascis Summary

HTML Summary

HTML(HyperText Markup Language) is used to define your page content, stcuture. You don't use it for styling purposes. Use CSS for that!

  • HTML uses "elements" to describe (annotate) content
  • HTML elements typically have an opening tag, content and then a closing tag
  • You can also have void (empty) elements like images
  • you can use HTML to create page and put your content

Learn more about HTML elements on the MDN HTML element reference

CSS Summary

CSS(Cascading Style Sheets) is used for styling your page content

  • Styles are assigned via property-value paris
  • You can assign styles via the "style" attribute
  • To avoid code duplication, you typically use global styles.
  • ALternately, you can work with external stylesheet files which you "link" to
  • When working with CSS, concepts like "inheritance", "specificity" and the "box model"
