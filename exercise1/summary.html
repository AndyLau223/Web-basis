HTML & CSS Basics Summary

  1. HTML Summary

  2. HTML (HyperText Markup Language) is used to define our page content, strucutre and meaning. You don't use it for styling purposes. Use Css for that instead!

    HTML uses "elements" to describe (annotate) content

    HTML elements typically have an opening age, content and then a closing tag

    You can also have void (empty) elements like images

    You can also configure elements with attributes

    There's a log list of elements but you will gain experience over time

    Learn more about all available HTML elements on thte MDN HTML reference

  3. CSS Summary

  4. CSS (Cascading Style Sheets) is used to style your web page.

    Style are assigned via property-value paris

    You can assign styles via the "style" attribute

    To avoid code duplication, you typically use global styles

    Alternally, you can work with external stylesheet files which you "link" to

    When working with CSS, concepts like "inheritance", "specificity" and the "box model" should be understood

    Learn more about all available CSS properties and values on the MDN CSS property reference
